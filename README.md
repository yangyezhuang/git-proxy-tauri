## Build Guidelines

### Prerequisites

* Rust (latest version)
* Node.js >= 16
* NPM >= 9


### Pull the Code

```bash
git clone https://github.com/yangyezhuang/git-proxy --depth=1
```

### Compile and Run

```bash
npm run tauri dev

npm run tauri build
```
