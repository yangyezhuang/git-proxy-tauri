## 构建项目

### 运行环境要求
* Ruts (latest version)
* Node.js >= 16
* NPM >= 9


### 拉取代码

```bash
git clone https://github.com/yangyezhuang/git-proxy --depth=1
```

### 编译运行开发版本

```bash
npm run tauri dev

npm run tauri build
```
